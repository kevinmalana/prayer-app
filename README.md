# Prayer App

Catholic prayer app built with Expo + TypeScript.

## Local testing

```bash
npm install
npm start
```

Then scan the QR code with Expo Go.

## Installable test builds

### Android
```bash
npm run eas:build:android
```

### iPhone
```bash
npm run eas:build:ios
```

These use EAS Build and generate installable preview builds for device testing.
